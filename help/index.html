1. Начало работы

Откройте страницу входа, введите логин и пароль, выданные администратором вашего совета. При первом входе примите условия обработки персональных данных.

Роли:

  • Администратор — полный доступ, удаление записей;
  • Модератор — создание и редактирование;
  • Просмотр — только чтение.

Данные вашего совета изолированы от других организаций в облачной системе (модель SaaS).

2. Входящий журнал

  1. Перейдите во вкладку «Входящий журнал».
  2. Нажмите «Добавить письмо» и заполните поля: дата, номер, отправитель, краткое содержание.
  3. Укажите срок ответа — система подсветит просроченные письма.
  4. При необходимости прикрепите скан (PDF, изображение).
  5. Используйте фильтры и поиск для быстрого доступа к записям.

3. Исходящий журнал

Исходящие письма регистрируются как ответы общественного совета. При создании можно связать запись с входящим письмом — это отображается в обоих журналах и в статистике.

4. Статистика и KPI

На вкладке «Статистика» отображаются KPI, графики динамики обращений и сводка по комиссиям. Данные обновляются при каждой загрузке страницы.

5. Экспорт и импорт

В верхней панели доступны экспорт в CSV и JSON. Импорт JSON — только для администраторов при миграции данных. Перед импортом сделайте резервную копию.

6. Мобильная версия

На экранах шириной до 768 px таблицы отображаются в виде карточек, внизу — навигация по разделам. Кнопка «Показать фильтры» открывает панель поиска.

Кіру бетінде әкімші берген логин мен құпиясөзді енгізіңіз. Бірінші кіргенде дербес деректерді өңдеу шарттарын қабылдаңыз.

2. Кіріс журналы

«Кіріс журналы» бөлімінде жаңа хатты тіркеңіз, жауап мерзімін көрсетіңіз, қажет болса сканерді тіркеңіз.
