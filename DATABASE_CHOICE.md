# 🗄️ БАЗЫ ДАННЫХ ГОТОВЫ К ЗАГРУЗКЕ

## 📊 Что было создано

### 2 SQL файла для разных БД:

1. **database.sql** (~50 KB)
   - Для MySQL 5.7+
   - Для MariaDB 10.2+
   - Рекомендуется для большинства хостингов

2. **database_pg.sql** (~50 KB)
   - Для PostgreSQL 10+
   - Если ваш хостинг использует PostgreSQL

## 🎯 Быстрый выбор

### Если вы не знаете, какую БД использовать:

**Выберите MySQL** (`database.sql`)
- Это самая популярная БД
- Поддерживается на 99% хостингов
- Проще в использовании

### Если вы знаете, что используется PostgreSQL:

**Выберите PostgreSQL** (`database_pg.sql`)
- Более мощная БД
- Лучше для больших объемов данных

## 📥 Как загрузить БД

### Способ 1: Через phpMyAdmin (самый простой)

1. Откройте phpMyAdmin на вашем хостинге
2. Нажмите "Импорт"
3. Выберите файл `database.sql`
4. Нажмите "Выполнить"

**Готово!** БД загружена и готова к использованию.

### Способ 2: Через SSH (для опытных)

```bash
# Для MySQL
mysql -u root -p os_journal < database.sql

# Для PostgreSQL
psql -U postgres -d os_journal -f database_pg.sql
```

### Способ 3: Через панель управления хостингом

- cPanel → phpMyAdmin → Импорт
- Plesk → Базы данных → phpMyAdmin → Импорт
- ISPmanager → Базы данных → phpMyAdmin → Импорт

## ✅ Проверка после загрузки

### 1. Проверить через phpMyAdmin:
- Откройте БД `os_journal`
- Проверьте, что все таблицы созданы
- Проверьте, что есть тестовые данные

### 2. Проверить через приложение:
- Откройте `http://ваш-хостинг.com/dashboard.html`
- Авторизуйтесь: `admin / password`
- Проверьте, что загружаются статистики

### 3. Проверить через API:
```bash
curl http://ваш-хостинг.com/api/statistics.php
```

Должен вернуться JSON с данными.

## 🔧 Конфигурация

После загрузки БД отредактируйте `.env.local`:

```bash

DB_PORT=3306
DB_USER=root
DB_PASS=ваш_пароль
DB_NAME=os_journal
```
